# GPGPU 自定义指令编码

所有自定义指令复用 RV32F opcode (`1010011`)，在 `funct7` 域区分功能组，`rs2` 域区分子功能。

## 编码格式 (funct7 分组)

```
 31       25 24     20 19     15 14  12 11      7 6        0
+-----------+---------+---------+------+---------+----------+
|  funct7   |   rs2   |   rs1   |  rm  |    rd   | 1010011  |
+-----------+---------+---------+------+---------+----------+
```

## 1. 科学函数 — funct7 = `0110000`

| 指令 | rs2[24:20] | `.word f0,f0` | 操作 |
|------|-----------|---------------|------|
| `fexp.s rd, rs1` | `00000` | `0x60000053` | `e^x` |
| `fln.s rd, rs1` | `00001` | `0x60100053` | `ln(x)` |
| `frcp.s rd, rs1` | `00010` | `0x60200053` | `1/x` |
| `frsqrt.s rd, rs1` | `00011` | `0x60300053` | `1/sqrt(x)` |
| `ftanh.s rd, rs1` | `00100` | `0x60400053` | `tanh(x)` |
| `fsigmoid.s rd, rs1` | `00101` | `0x60500053` | `1/(1+e^{-x})` |
| `fsin.s rd, rs1` | `00110` | `0x60600053` | `sin(x)` |
| `fcos.s rd, rs1` | `00111` | `0x60700053` | `cos(x)` |

**指令字计算示例** (`fexp.s f15, f0`):
```
funct7=0110000 rs2=00000 rs1=00000 rm=000 rd=01111
→ 0x600007D3
```

**通用编码公式**:
```
(0x60000053) | (rd << 7) | (rs1 << 15) | (rm << 12)
```
rs2 已固定为各指令对应的值（0~7），不需要额外偏移。

## 2. 低精度浮点转换

### BF16 — funct7 = `0100010`

| 指令 | rs2[24:20] | `.word f0,f0` | 操作 |
|------|-----------|---------------|------|
| `fcvt.s.bf16 rd, rs1` | `00000` | `0x44000053` | bf16→f32 |
| `fcvt.bf16.s rd, rs1` | `00001` | `0x44200053` | f32→bf16 |

### E4M3 — funct7 = `0100100`

| 指令 | rs2[24:20] | `.word f0,f0` | 操作 |
|------|-----------|---------------|------|
| `fcvt.s.e4m3 rd, rs1` | `00000` | `0x48000053` | e4m3→f32 |
| `fcvt.e4m3.s rd, rs1` | `00001` | `0x48200053` | f32→e4m3 |
| `fcvt.s.e5m2 rd, rs1` | `00010` | `0x48400053` | e5m2→f32 |
| `fcvt.e5m2.s rd, rs1` | `00011` | `0x48600053` | f32→e5m2 |

### E2M1 — funct7 = `0100110`

| 指令 | rs2[24:20] | `.word f0,f0` | 操作 |
|------|-----------|---------------|------|
| `fcvt.s.e2m1 rd, rs1` | `00000` | `0x4C000053` | e2m1→f32 |
| `fcvt.e2m1.s rd, rs1` | `00001` | `0x4C200053` | f32→e2m1 |

## 编码空间总览

```
funct7[31:25]  用途
─────────────────────────────────
0000000         RV32F 标准 (fadd/sub/mul 等)
0000100         RV32F 标准 (fsub_s)
0001000         RV32F 标准 (fmul_s)
0001100         RV32F 标准 (fdiv_s)
0010000         RV32F 标准 (fsgnj 系列)
0010100         RV32F 标准 (fmin/fmax)
0100010         自定义 LP float: bf16 转换
0100100         自定义 LP float: e4m3/e5m2 转换
0100110         自定义 LP float: e2m1 转换
0101100         RV32F 标准 (fsqrt_s)
0110000         自定义 科学函数: exp/ln/rcp/rsqrt/tanh/sigmoid/sin/cos
1010000         RV32F 标准 (feq/flt/fle)
1100000         RV32F 标准 (fcvt.w.s/wu.s)
1101000         RV32F 标准 (fcvt.s.w/wu)
1110000         RV32F 标准 (fmv.x.w/fclass.s)
1111000         RV32F 标准 (fmv.w.x)
─────────────────────────────────
未使用: 0110100, 0111000, 0111100, 1000000-1001100, 1011000-1011100
```

## 在汇编中使用

```asm
# 科学函数 — 必须用 .word 手工编码（汇编器不认识这些助记符）
.word 0x60000053        # fexp.s     f0, f0
.word 0x60100053        # fln.s      f0, f0
.word 0x60200053        # frcp.s     f0, f0
.word 0x60300053        # frsqrt.s   f0, f0
.word 0x60400053        # ftanh.s    f0, f0
.word 0x60500053        # fsigmoid.s f0, f0
.word 0x60600053        # fsin.s     f0, f0
.word 0x60700053        # fcos.s     f0, f0

# LP float 转换
.word 0x44000053        # fcvt.s.bf16  f0, f0
.word 0x44200053        # fcvt.bf16.s  f0, f0
```
